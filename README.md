### Prerequisites

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/swasthify.git
   cd Swasthify-
   ```

2. **Install frontend dependencies**
   ```bash
   cd frontend
   npm install
   ```

